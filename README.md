# Inventory Management System

A sleek and interactive inventory management dashboard built with **Streamlit** and **Python**, with **MySQL** integration for storing and managing data. Custom **CSS styling** gives it a modern look with gradient cards, button-style input fields, and stylish buttons.

## Features
- Manage inventory with an easy-to-use dashboard.
- Interactive cards and inputs styled like buttons.
- Blue gradient UI for a modern, clean design.
- MySQL integration for secure and efficient data storage.

## Tech Stack
- **Python** – Core logic and functionality.
- **Streamlit** – Web application framework for interactive dashboards.
- **CSS** – Custom styling for UI enhancements.
- **MySQL** – Database management.
